# Overview

AI Design Review is an intelligent UI/UX auditing platform that leverages **Google Gemini AI** to evaluate user interface designs and generate detailed design insights.

It helps designers and developers identify usability issues, accessibility problems, visual inconsistencies, and UX improvements while providing practical recommendations based on modern design principles.

---

# Features

- AI-powered UI/UX analysis
- Visual design evaluation
- Accessibility auditing
- Responsive design assessment
- Detailed scoring dashboard
- UX heuristic evaluation
- Actionable design recommendations
- Fast React + Vite frontend
- Express backend with Gemini integration
- Professional audit reports

# Getting Started

## Prerequisites

- Node.js 18+
- npm

---

## Installation

Clone the repository

```bash
git clone <your-repository-url>
cd designreview-ai
```

Install dependencies

```bash
npm install
```

---

# Environment Variables

Create a `.env.local` (or `.env`) file in the project root.

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

Get your Gemini API Key from Google AI Studio.
